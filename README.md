## GEN
generate project from scaffold

## REQUIREMENTS
- sys: unix*
- jdk8+
- maven3.3+


## RUN (in bash)

```
git clone git@github.com:guxingke/gen.git
cd gen
.bin/package.sh
.bin/install.sh
```

## USE
```bash
gg 

gg info

gg list

gg demo info 

gg init demo test # be gen test from demo scaffold into current dir

```
